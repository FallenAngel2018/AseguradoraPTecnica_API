# API Aseguradora - Práctica Técnica

Sistema de gestión para aseguradora desarrollado con ASP.NET Core y SQL Server.

## Requisitos Previos

- Visual Studio 2022
- SQL Server
- .NET Framework 4.8 o .NET 8

## Tecnologías Utilizadas

- **Backend:** ASP.NET MVC 5 / ASP.NET Core
- **Frontend:** Razor Pages / jQuery + Bootstrap
- **Base de Datos:** SQL Server

## Instalación y Configuración

### 1. Descargar el Proyecto

Descarga el archivo ZIP del proyecto y descomprímelo en tu directorio de trabajo.

### 2. Abrir la Solución

Abre el archivo `AseguradoraPTecnica.sln` con Visual Studio 2022.

### 3. Configurar la Conexión a la Base de Datos

Navega al archivo de configuración:

"ConnectionStrings": {
  "DefaultConnection": "Server=localhost;Database=AseguradoraBD;User Id=usuario;Password=clave;Encrypt=false;"
}

### 4. Ejecutar la Aplicación

Presiona `F5` o haz clic en el botón "Run" en Visual Studio para iniciar la aplicación.
